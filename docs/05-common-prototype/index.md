# 공통 프로토타입

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00001030

## 개요

공통 프로토타입은 무인정보단말기 대분류 유형별(유통, 주문, 발권, 안내)로 제공되는 UI 프로토타입 목록이다. 기본, 고대비, 낮은 화면 등 다양한 모드별 UI 예시를 제공한다.

## 상세 내용

### 프로토타입 분류

프로토타입은 다음 세 가지 모드로 제공된다.

| 모드 | 설명 |
|------|------|
| 기본 | 일반적인 무인정보단말기 UI |
| 고대비 | 시각적 접근성을 위한 고대비 모드 UI |
| 낮은 화면 | 낮은 높이의 화면을 위한 UI (휠체어 사용자 등) |

### 프로토타입 목록 (유통형 - 1페이지 기준, 전체 21건)

| 번호 | 유형 | 제목 | 모드 |
|------|------|------|------|
| 1 | 유통형 | 무인주유기 기본 UI | 기본 |
| 2 | 유통형 | 무인주유기 고대비 UI | 고대비 |
| 3 | 유통형 | 무인주유기 낮은 화면 UI | 낮은 화면 |
| 4 | 유통형 | 무인주차정산기 기본 UI | 기본 |
| 5 | 유통형 | 무인주차정산기 고대비 UI | 고대비 |
| 6 | 유통형 | 무인주차정산기 낮은 화면 UI | 낮은 화면 |
| 7 | 유통형 | 무인결제기 기본 UI | 기본 |
| 8 | 유통형 | 무인결제기 고대비 UI | 고대비 |

> 전체 21건, 3페이지로 구성
