# 장애 유형별 공통 UI 프로토타입

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00001032

## 개요

장애 유형별 공통 UI 프로토타입은 시·청각 장애인이 무인정보단말기를 실제로 사용하는 과정을 영상(음성 자막 포함)으로 제공한다. 키오스크 유형별(유통형·주문형·발권형·안내 및 기타)로 음성 안내, 점자, 수어 안내 프로토타입이 구성되어 있다.

> 참고: 해당 메뉴 접근 시 첫 번째 하위 페이지(유통형 > 음성)로 자동 이동됩니다.

## 상세 내용

### 구성 체계

| 키오스크 유형 | 음성 | 점자 | 수어 |
|--------------|------|------|------|
| 유통형 | O | O | O |
| 주문형 | O | O | O |
| 발권형 | O | O | O |
| 안내 및 기타 | O | O | O |

### 프로토타입 예시 — 유통형 음성 안내 (무인 결제기)

아래는 무인 결제기 사용 교육 영상의 음성 안내 자막 내용이다.

#### 시작 안내

- 무인결제기입니다.
- 화면 보기가 어렵다면 오른쪽 아래에 접근성 버튼을 눌러 음성 안내를 시작합니다.
- 상품 결제는 전체 4단계: 첫 번째 상품 입력, 두 번째 포인트 적립, 세 번째 결제하기, 네 번째 완료입니다.

#### 1단계 — 상품 입력

- 결제하실 상품의 바코드를 바코드 인식기로 인식시킵니다.
- 바코드 인식기는 단말기의 아래에 있으며 바코드는 상품 포장에 인쇄·부착되어 있습니다.
- 상품 입력 시 상품 내용이 음성으로 안내됩니다.
- 입력 취소는 오른쪽 가위 버튼, 상품 결제는 오른쪽 아래 결제하기 버튼을 누릅니다.

#### 2단계 — 포인트 적립

- 예상 적립금액이 안내됩니다.
- 적립하지 않으시면 왼쪽 아니오 버튼, 적립하시면 오른쪽 예 버튼을 누릅니다.
- 예 버튼 선택 시 휴대전화 번호 입력을 시작합니다.

#### 3단계 — 결제하기

- 결제 방법: 신용카드, NIA 페이, 쿠폰 사용, 바코드 할인, 기프트 카드 (총 5가지)
- 결제 방법 버튼을 누르시면 결제를 시작합니다.
- 신용카드 결제 시 카드 인식기에 카드의 앞면을 위로 향하여 끝까지 넣습니다.

#### 4단계 — 완료

- 정상 결제 후 카드가 배출됩니다.
- 영수증 출력 여부를 선택합니다.
- 구매하신 상품, 카드, 이어폰 등 잊으신 물건을 확인합니다.
- 5초 뒤 모든 사용 설정이 초기화됩니다.
