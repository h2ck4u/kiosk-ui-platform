# 체험

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00000914

## 개요

무인정보단말기 체험 섹션은 두 가지 하위 메뉴로 구성됩니다.

1. **무인정보단말기 교육 동영상** — 다양한 유형의 키오스크 UI 교육용 동영상 제공
2. **무인정보단말기 화면 체험** — 실제 키오스크 화면을 직접 체험할 수 있는 외부 서비스 (https://www.kioskux.com)

## 상세 내용

### 무인정보단말기 교육 동영상

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00000780

총 **85건**의 교육 동영상이 제공되며, 유형별로 분류됩니다.

#### 동영상 유형 분류

| 유형 | 설명 |
|------|------|
| 기본 | 일반 화면 모드의 키오스크 UI |
| 고대비 | 고대비 모드의 키오스크 UI |
| 낮은 화면 | 낮은 화면(저시력·휠체어) 모드의 키오스크 UI |
| 기타 | 그 외 유형 |
