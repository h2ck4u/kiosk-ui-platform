# 공통 UI 가이드

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00001029

## 개요

누구에게나 동등한 기회 제공을 위해 무인정보단말기의 사용성 및 접근성을 보장할 필요가 있다. 무인정보단말기의 공통 UI 가이드 및 대분류 UI 가이드를 참고하여, 모든 국민이 전반적으로 무인정보단말기의 접근과 활용에 어려움을 겪지 않도록 해야 한다.

## 상세 내용

### 무인정보단말기 대분류 UI 가이드

#### 목적

**누구에게나 동등한 기회 제공을 위해 무인정보단말기의 사용성 및 접근성을 보장할 필요가 있다.**

무인정보단말기의 공통 UI 가이드 및 대분류 UI 가이드를 참고하여, 모든 국민이 전반적으로 무인정보단말기의 접근과 활용에 어려움을 겪지 않도록 해야 한다.

#### 설명

본 가이드는 국내외 무인정보단말기 관련 법령 및 표준/가이드 리서치, 16가지 중분류(64종)에 대한 고객 여정 및 UI 분석, 비장애인과 고령자 대상 FGI, 장애인 대상 FGI, 접근성 전문가 인터뷰를 통해 도출된 무인정보단말기 UI 원칙을 기반으로 제작했다.

**UI 가이드 도출 과정**

1. 국내외 무인정보단말기 관련 법령 및 표준/가이드 리서치
2. 16가지 중분류(64종)에 대한 고객 여정 및 UI 분석
3. 비장애인과 고령자 대상 FGI
4. 장애인 대상 FGI
5. 접근성 전문가 인터뷰
6. **UI 원칙 도출** → 공통 UI 가이드 / 대분류 UI 가이드 / 장애 유형별 UI 가이드

#### 구성

무인정보단말기 UI 가이드는 공통 UI 가이드 및 대분류 UI 가이드(유통, 주문, 발권, 안내 및 기타)로 구성되어 있다. 무인정보단말기의 효과적인 사용 경험을 제공하기 위한 인터페이스 설계 방법을 제공한다.
