# UI 리소스

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00000915

## 개요

무인정보단말기 UI 플랫폼에서 제공하는 공통 UI 컴포넌트 리소스 목록이다. 각 버튼 및 UI 요소는 기본(Normal)과 고대비(High Contrast) 두 가지 모드로 제공되며, 개별 다운로드가 가능하다.
