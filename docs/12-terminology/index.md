# 무인정보단말기 용어 가이드

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00000918

## 개요

무인정보 단말기 용어 가이드는 통일된 용어 및 문구를 제공하여 사용자의 혼란을 최소화하고 무인정보 단말기 사용의 효율성을 높이는 데 도움이 될 수 있다. 무인정보 단말기 용어 가이드를 참고하여 일관된 용어 사용을 권장한다.

## 상세 내용

### 구성

용어 가이드는 단말기 유형별 대분류로 구성되며, 총 **35건**의 용어 항목을 제공한다.

#### 유통

- 무인주유기
- 무인주차정산기
- 무인결제기
- 무인도서대여반납기

#### 주문

- 무인주문기

#### 발권

- 무인민원발급기
- 무인발권기
- 무인발매기
- 무인증명발매기
- 셀프체크인
- 무인처방전발매기
- 금융자동화기기

#### 안내 및 기타

- 종합정보시스템
- 위치정보시스템
- 무인사용자인증기
- 기타(사물함, 택배함 등)
