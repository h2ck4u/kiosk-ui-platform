# 발권형

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00001038

## 개요

발권형 무인정보단말기는 티켓이나 인증서, 수납증 등을 발급하거나 출력하는 기능을 갖춘 자동화된 시스템을 의미한다. 여러 업종에서 사용되며, 예매, 입장권 발급, 수납증 발행 등의 작업을 간편하게 처리할 수 있도록 도와준다.

## 상세 내용

### 공통 프로세스 흐름

| 단계 | 설명 |
|------|------|
| 접근 | 사용자가 단말기에 접근 |
| 1. 시작안내 | 사용법 및 사용범위 안내, 사용환경 설정 |
| 2. 기본선택 | 출력 문서 또는 서비스/콘텐츠 선택 |
| 3. 본인확인 | 개인정보 문서 발급 시 본인 인증 |
| 4. 옵션선택 | 문서 내용 설정 또는 서비스 옵션 선택 |
| 5. 내역확인 | 발권 내역 및 결제 금액 확인 |
| 6. 결제하기 | 결제 수단 선택 및 결제 |
| 7. 발권 | 서류 및 티켓 출력 |
| 8. 완료 | 서비스 완료 신호 제공 |
| 이탈 | 프로세스 중단 |

---

### 시작안내

#### 예시 화면

![시작안내 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process3_01.jpg)

#### 프로세스 분석

사용자에게 시작화면을 통해 사용 직전 무인정보단말기의 사용법과 사용범위를 알려주고 사용환경을 설정할 수 있도록 도와준다.

---

### 기본선택

#### 예시 화면

![기본선택 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process3_03.jpg)

#### 프로세스 분석

사용자가 출력을 원하는 문서의 선택과 예약, 예매 및 결제하여 탑승할 교통수단, 관람할 문화콘텐츠, 진료받을 의료서비스 등을 선택하기 위한 화면이다.

---

### 본인확인

#### 예시 화면

![본인확인 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process3_05.jpg)

#### 프로세스 분석

발권형 무인정보단말기의 특성상 단순 예약, 예매 등을 제외하면 개인정보가 담긴 문서 발급의 경우 본인확인 단계가 필요하다.

---

### 옵션선택

#### 예시 화면

![옵션선택 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process3_07.jpg)

#### 프로세스 분석

출력을 원하는 문서의 내용을 설정하거나 예약, 예매 및 결제할 서비스와 콘텐츠 상품의 옵션을 선택할 수 있는 화면이다.

---

### 내역확인

#### 예시 화면

![내역확인 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process3_09.jpg)

#### 프로세스 분석

발권 내역, 옵션 선택 내용, 최종 금액 등 결제 금액의 정산을 종합적으로 안내하는 화면이다.

---

### 결제하기

#### 예시 화면

![결제하기 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process3_11.jpg)

#### 프로세스 분석

내역확인 이후 무인정보단말기가 지원하는 결제 방법을 제공하는 화면이다. 결제 직전이므로 총 결제 금액을 같이 제공한다.

---

### 발권

#### 예시 화면

![발권 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process3_14.jpg)

#### 프로세스 분석

발권형 무인정보단말기의 특성상 서류와 티켓의 출력이 필요하다. 그에 따라 출력의 진행 및 완료 상태를 명확히 알려줘야 한다.

---

### 완료

#### 예시 화면

![완료 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process3_16.jpg)

#### 프로세스 분석

안내형 무인정보단말기를 제외한 모든 단말기는 완료화면을 가진다. 프로세스의 진행은 목표를 가지고 있고 목적 달성 시 사용자에게 명확한 서비스 완료 신호를 제공한다.
