# 안내 및 기타

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00001039

## 개요

안내형 무인정보단말기는 주로 고객에게 정보를 제공하거나 안내 역할을 하는 자동화된 시스템이다. 이러한 무인정보단말기는 다양한 업종에서 사용되며, 고객이 필요한 정보를 쉽게 얻을 수 있도록 도와주는 데 중점을 둔다.

## 상세 내용

### 공통 프로세스 흐름

| 단계 | 설명 |
|------|------|
| 접근 | 사용자가 단말기에 접근 |
| 1. 안내 | 시작화면을 통해 사용법 및 사용범위 안내 |
| 2. 세부안내 | 상세 정보 탐색 및 안내 |
| 이탈 | 프로세스 중단 |

### 안내

#### 예시 화면

![안내 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process4_01.jpg)

#### 프로세스 분석

사용자에게 시작화면을 통해 사용 직전 무인정보단말기의 사용법과 사용범위를 알려주고 사용환경을 설정할 수 있도록 도와준다.

메뉴의 구성이 복잡한 경우가 많아 서비스 사용자들이 쉽게 알아볼 수 있도록 효과적인 정보표시법을 사용해야 한다. 일정 시간 동안 입력이 없을 때 처음으로 가는 기능이 필요하다.
