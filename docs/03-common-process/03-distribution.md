# 유통형

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00001036

## 개요

유통형 무인정보단말기는 주로 소매업 및 유통 업계에서 사용되는 자동화된 기기로, 사용자가 제품을 검색하고 구매하는 프로세스를 간편하게 만들어주는 시스템이다. 주로 상품코드(바코드 등)를 사용자가 직접 스캔해서 제품을 구매하는 유형이다.

## 상세 내용

### 공통 프로세스 흐름

| 단계 | 설명 |
|------|------|
| 접근 | 사용자가 단말기에 접근 |
| 1. 시작안내 | 사용법 및 사용범위 안내, 사용환경 설정 |
| 2. 기본선택 | 반출할 상품 등록 |
| 3. 내역확인 | 상품 목록 및 결제 금액 확인 |
| 4. 결제하기 | 결제 수단 선택 및 결제 |
| 5. 완료 | 서비스 완료 신호 제공 |
| 이탈 | 프로세스 중단 |

---

### 시작안내

#### 예시 화면

![시작안내 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process1_01.jpg)

#### 프로세스 분석

사용자에게 시작화면을 통해 사용 직전 무인정보단말기의 사용법과 사용범위를 알려주고 사용환경을 설정할 수 있도록 도와준다.

---

### 기본선택

#### 예시 화면

![기본선택 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process1_03.jpg)

#### 프로세스 분석

서비스 특징에 따라 무인정보단말기 근처로 소비자가 직접 상품을 운반하여 반출할 상품을 등록시키기 위한 화면이다.

---

### 내역확인

#### 예시 화면

![내역확인 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process1_06.jpg)

#### 프로세스 분석

상품 등록의 완료에 따라 반출할 상품 목록의 상태와 상품 금액, 할인 금액, 최종 금액 등 결제 금액의 정산을 종합적으로 안내하는 화면이다.

---

### 결제하기

#### 예시 화면

![결제하기 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process1_08.jpg)

#### 프로세스 분석

내역확인 이후 무인정보단말기가 지원하는 결제 방법을 제공하는 화면이다. 결제 직전이므로 총 결제 금액을 같이 제공한다.

---

### 완료

#### 예시 화면

![완료 예시 화면](https://www.kioskui.or.kr/cmsh/kioskui.or.kr/images/content/img_process1_10.jpg)

#### 프로세스 분석

안내형 무인정보단말기를 제외한 모든 단말기는 완료화면을 가진다. 프로세스의 진행은 목표를 가지고 있고 목적 달성 시 사용자에게 명확한 서비스 완료 신호를 제공한다.
