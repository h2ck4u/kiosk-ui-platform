# 배리어프리 키오스크 UI 가이드

> 출처: https://www.kioskui.or.kr/index.do?menu_id=00001450

## 개요

무인으로 운영되는 키오스크를 위한 배리어프리 키오스크 UI 가이드를 통해 **업종과 서비스 종류에 상관없이 일관된 사용자 경험과 접근성 UX 품질을 확보**한다.

## 상세 내용

### 목표

#### 일관된 접근성과 사용성 확보

- 통합된 접근성 기준 수립
- 직관적 인터페이스 제공
- 일관된 사용 경험 제공

#### 유연하고 체계적인 디자인 시스템 구축

- 모듈형 키오스크 디자인 시스템 개발
- 미래 확장성과 유지관리 용이한 설계
- 표준화된 컴포넌트 제공

#### 서비스별 파생 모델 개발 기반 마련

- 분야별 특성을 반영한 파생 모델 구성
- 맞춤형으로 확장 가능한 구조 설계
- 민간 활용도를 높이는 파생 전략 마련

### 내용

- 대표 과업 기반의 사용자 흐름과 시각 구조 설계
- 서비스 별 레이아웃 스타일과 인터랙션 스타일 정의
- 디자인 가이드를 통해 현장 적용 가능한 표준 UI/UX 10종 개발
- 일관된 사용성을 통해 익숙하고 학습성 높은 UX 품질 확보
- 접근성 검증 기준에 충족하는 UI/UX 적용

### 구성

소분류 10종에 대한 표준 UI 가이드로 구성한다.

| 대분류 | 중분류 | 소분류 |
|--------|--------|--------|
| 유통형 | 무인주유기 | 전기충전소 무인충전기 |
| 유통형 | 무인결제기 | 무인판매점(아이스크림할인점) |
| 유통형 | 무인반납기 | 일회용컵 간이 반납기 |
| 유통형 | 무인우편접수기 | - |
| 주문형 | 무인주문기 | 무인카페 |
| 주문형 | 스마트자판기 | - |
| 발권형 | 금융자동화기기 | CD/ATM |
| 안내형/기타 | 종합안내시스템 | 관광안내키오스크 |
| 안내형/기타 | 기타 | 사물함 키오스크 |
| 안내형/기타 | 기타 | 빨래방 키오스크 |
